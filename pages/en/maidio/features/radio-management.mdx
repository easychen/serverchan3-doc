# Radio Management

## Create Radio

### Basic Information
- Radio Name: Choose an easily identifiable name for your radio
- Feed Sources: Add RSS feed sources, supporting the following methods:
  - Manually enter RSS URL
  - Import OPML file

### Feed Source Management
- Add: Click the "+" button to add new feed source
- Edit: Swipe left on the feed item and click "Edit"
- Delete: Swipe left on the feed item and click "Delete"

## Edit Radio

- Modify Name: Click on the radio name to edit
- Manage Subscriptions:
  - Add/Remove feed sources
  - Enable/Disable feed sources

## Delete Radio

1. Swipe left on the radio item in the main interface
2. Click the "Delete" button
3. Confirm deletion operation
