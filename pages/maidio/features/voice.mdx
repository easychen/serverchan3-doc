# 语音设置

本应用支持本地和云端两种语音合成方式，您可以根据需要选择合适的语音方案。

## 本地语音

本地语音使用设备自带的语音引擎，无需联网即可使用。

### Android 系统设置
在使用本地语音之前，请先完成以下准备工作：

1. 安装中文TTS以及语音包

通常来讲，手机系统会自带一个TTS引擎，但这只有一种声音，不适合对话场景。因此，我们需要额外再安装一个TTS引擎。

   - 进入系统设置
   - 找到语言和输入法设置
   - 查看TTS引擎，如果只有一个，则需要安装一个TTS引擎

2. 推荐使用以下语音引擎：
   - 小米语音引擎（可从[这里](https://www.52pojie.cn/thread-1683594-1-1.html)下载，第三方软件，请自行判定风险）
   - Google文字转语音引擎（可从[Google Play](https://play.google.com/store/apps/details?id=com.google.android.voiceservices)下载）

   * 注意，小米语音引擎虽然可以离线使用，但默认会联网以获得更好的效果。不收费，但有可能产生流量费用。

### iOS 系统
iOS系统自带中文语音引擎，无需额外设置。

### 语音参数设置
您可以调整以下参数来获得最佳的语音效果：

- 语速：调整语音播放的快慢
- 音调：调整语音的音高
- 音量：调整语音的大小

## 云端语音

云端语音使用AI技术生成更自然的语音效果，需要联网使用。

### 使用前准备
1. 在设置界面填写API Key
   - iOS用户：使用 SiliconFlow.cn 的 API Key
   - Android用户：使用 Api2d.com 的 API Key

2. 开启"使用云端TTS"选项

### 语音参数设置
云端语音同样支持调整：
- 语速
- 音调
- 音量

## 人设定制

您可以为主持人和助手分别设置不同的语音特征和个性化设置。

### 主持人/助手设置
- 设置名称：为角色取一个合适的名字
- 性格特点：定制角色的性格特征
- 表达风格：调整说话的语气和风格

## 优化设置

在设置界面，您还可以开启以下功能来优化语音效果：

- 二次润色：使用AI对生成的对话进行优化，使其更自然流畅
- 自动翻译：可自动翻译非中文内容（如果需要）
