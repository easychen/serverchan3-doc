# 安装

## 商店安装和手动安装

请进入[官网下载页面](https://sc3.ft07.com/client)：
- 如果有手机品牌对应的市场，可以点击对应的按钮到市场下载或者搜索「Server酱」；
- 如果没有，则直接下载APK后手动安装。

<Callout>
市场的安装包和直接下载的安装包是完全一致的。一般因为审核问题，市场可能比最新版本晚一两天
</Callout>

## 配置

<Callout type="info">
推荐使用后台驻留方式，支持所有的Android品牌，信息秒达。
<br/>后续有省电需求或者老被系统杀后台再采用无后台方式。
</Callout>


请依次打开以下权限：

- [x] 通知权限，特别是「通知分类」中「用户私有通知」的权限，部分手机可指定提醒方式和声音
- [x] 通知过滤规则：全部设为重要
- [x] 后台常驻/锁屏保护
- [x] 自启动/自动后台
- [x] 后台弹出界面
- [x] 应用省电设置：无限制

### 无后台推送

当应用不在后台时，只有通过厂商通道才能收到消息。本应用支持以下厂商通道：

- `iOS`：苹果手机
- `FCM`：支持谷歌推送的手机，注册设备需要海外网络，国内大部分网络可以接受推送
- `MiPush`：小米、红米系列手机
- `Push Kit`：华为手机和平板，EMUI 3.0及以上，HMS Core 3.0.0及以上
- `HONOR Push`：荣耀手机和平板，国内Magic UI 4.0及以上
- `OPPO Push`：支持 ColorOS3.1及以上的系统的OPPO的机型，一加5/5t及以上机型，realme所有机型
- `vivo Push`：vivo、iQOO系列手机
- `Flyme Push`：魅族手机，Flyme系统
